    Safe Travels!

Data is an asset and securing it is everyone's job. It is important to know how to protect data and information in times of threat and how to identify vulnerabilities in the system. Regardless of their role, every employee in an organization must know the general security guidelines and adapt them. Here's what you can do to secure data:

  • Update your password every 90 days and use combination of letters and numbers.
  • Use secure platforms for file transfers.
  • Use discretion when typing in passwords if you're not alone.
  • Store confidential data and files in secure storage and on authorized cloud platforms only.
  • Check access levels on a monthly basis to eliminate threats posed by unwanted access.
  • Use caution when opening attachments and do not open if the sender is from unknown origin.
  • Use different credentials for banking, social media and work accounts.
  • Use only authentic vendor sites and servers to download software or files.
  • Always update your computer and laptop to the latest version of the software release and apply patches whereever necessary as new versions eliminate known vulnerabilities.

While it seems basic and mostly common sense, these small mistakes can cost a fortune to the business. Please don't:

  • Respond to emails and attachments from unknown sources. If you have opened an unwanted attachment, please inform IT immediately so they analyze the risk.
  • Give away confidential passwords and login details to anyone, remember that IT never requests you to send passwords for verification.
  • Use public computers for office purpose. Restrict to using only a secure machine provided by the employer.
  • Leave sensitive data on your desk, shred the printouts after use.
  • Leave your computer unlocked, lock them down even if it is just for a quick restroom break.
  • Install applications from unknown sources, report them to IT if you really need an application and they will help you install the safest version.
  • Plug-in personal devices without IT authorization.


Report Phishing


Phishing is a malicious attempt to obtain your passwords, credentials and other secure information. This is mostly done through emails. Beware of:

  • Free offers and coupons from unknown emails.
  • Tax refund checks - IRS never asks for personal information over email.
  • Exclusive access to pre-release versions of software or games.
  • Emails from unknown people with attachments.
  • Emails claiming to come from IT requesting password and usernames but the actual person is not related IT.
In all of the above mentioned events, please report to IT helpdesk about the sender and if possible, attach a screenshot of the email itself. Do not ever open an the attachment from suspicious emails.
